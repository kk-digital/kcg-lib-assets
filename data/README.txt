Test Data for Decompression Testing
====================================

This folder contains test image files for benchmarking decompression performance.

Contents:
- png/ - PNG image files (696 files) for PNG decompression testing
- jpg/ - JPEG image files (14 files) for JPEG decompression testing

Files are named with sequential prefixes for uniqueness:
- Format: NNNN_png_[original_name].png
- Format: NNNN_jpg_[original_name].jpg

Source: Copied from kcg-game assets (flattened directory structure)

Purpose: Decompression benchmark testing
